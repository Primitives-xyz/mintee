# Introduction

Welcome to Mintee! This is the official documentation for the Mintee API and `Mintee.js`.

## What is Mintee?

A Solana NFT API built for speed from the ground up. We provide the easiest
way to ship integrations with Solana NFTs at a low monthly cost.

Beyond getting perfomament minting, we built an edge caching layer to
ensure you get data about an NFT faster than anyone else.
